Связь с **капитаном**

**Описание:**
Консольное Python-приложение для базового аудита безопасности Linux (Kali/Debian/Ubuntu).  Что проверяет и делает: права доступа, сеть, пакеты/версии, простые рекомендации. Сохраняет отчет в "report.json".


**Требования к системе:**

-OS: Linux (Kali/Debian/Ubuntu)

-Python: 3.10+ (обычно подойдет любой Python 3)

-Встроенные команды: "find", "ss", "dpkg-query" (на Kali обычно есть)

**Зависимости:**

Внешних Python-зависимостей нет (см. "requirements.txt").

Рекомендуется запускать с правами "sudo", иначе часть проверок может быть неполной!:

```bash

```

# Также можно без "sudo":

```bash
python3 audit.py
```

**Что проверяет:**
- права доступа к файлам и директориям (777, 666, sticky‑bit, world‑writable);
- наличие секретов (пароли, ключи) в общедоступных файлах;
- открытые сетевые порты и опасные сервисы (Telnet, FTP, SMB, MySQL);
- установленные сетевые пакеты (dpkg);
- версию ядра;
- настройки SSH (`PermitRootLogin`, `PasswordAuthentication` и др.);
- правила sudoers (NOPASSWD).

**Отчет:**

-Консоль: краткий список проблем + рекомендации

-JSON: "report.json"

**Тестирование** производилось на Kali Linux (чистая + уязвимая).
